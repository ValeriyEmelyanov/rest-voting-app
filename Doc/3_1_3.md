**3.1.3. Get user**
----
Returns json data about user.
* **URL** 
```
  /users/:id
```
* **Method:**
```
  GET
```
 
* **Headers:**
```
  Accept: application/json
  Authorization: <SecretToken>
```
* **URL Params:**
```
    id=[integer]
```
* **Success Response:**
```
    Code: 200 OK
    Body:
    {
        "id": 101,
        "name": "User",
        "email": "user@yandex.ru",
        "registered": "2019-09-17T17:41:30.312+0000",
        "enabled": true
    }
``` 
* **Error Response:**
```
    Code: 403 Forbidden
```
    or
```
    Code: 400 Bad Request
```

