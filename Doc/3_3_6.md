**3.3.6. Delete menu**
----
Removes menu. No returns.
* **URL** 
```
  /menus/:id
```
* **Method:**
```
  DELETE
```
 
* **Headers:**
```
  Accept: application/json
  Authorization: <SecretToken>
```
* **URL Params:**
```
    id=[integer]
```
* **Success Response:**
```
    Code: 204
    Body:
    [
    ]
```
