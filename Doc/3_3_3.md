**3.3.3. Get menu**
----
Returns json data about menu.
* **URL** 
```
  /menus/:id
```
* **Method:**
```
  GET
```
 
* **Headers:**
```
  Accept: application/json
  Authorization: <SecretToken>
```
* **URL Params:**
```
    id=[integer]
```
* **Success Response:**
```
    Code: 200 OK
    Body:
    [
    ]
```
