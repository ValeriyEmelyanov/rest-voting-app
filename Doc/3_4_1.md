**3.4.1. Get votes list by date**
----
Returns json data about votes list by date.
* **URL** 
```
  /votes/date/:date?page=:page&limit=:limit
```
* **Method:**
```
  GET
```
 
* **Headers:**
```
  Accept: application/json
  Authorization: <SecretToken>
```
* **URL Params:**
```
    date=[date]
    page=[integer]
    limit=[limit]
```
* **Success Response:**
```
    Code: 200 OK
    Body:
    [
    ]
```
