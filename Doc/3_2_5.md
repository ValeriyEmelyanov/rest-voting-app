**3.2.5. Update restaurant**
----
Updates restaurants and returns json data about updated restaurant.
* **URL** 
```
  /restaurants/:id
```
* **Method:**
```
  PUT
```
 
* **Headers:**
```
  Accept: application/json
  Authorization: <SecretToken>
```
* **URL Params:**
```
    id=[integer]
```
* **Success Response:**
```
    Code: 200
    Body:
    [
    ]
```
